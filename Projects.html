Project Overview

ADAPTLEARN AI is an advanced learning platform that leverages artificial intelligence to deliver personalized education experiences. COLLEGE-CENTRIC FOOD ORDERING

Web

Project Overview

This web application bridges local food vendors with college students and staff, creating a seamless food ordering and delivery ecosystem. The platform enables college communities to browse menus, place orders, and arrange deliveries to the college gate.

For shop owners, the system provides valuable data to predict order volumes, reduce food waste, optimize resources, and expand their customer base without significant infrastructure investments.

Key Features

  • Easy menu browsing and ordering
  • Delivery coordination to college gate
  • Order prediction analytics
  • Resource optimization tools
  • User-friendly vendor dashboard
  • Order status tracking

DIAFLOW

Web AI

Project Overview

Diaflow is an innovative AI-powered diagramming tool built with React, drawing inspiration from platforms like draw.io and mindmap.ai. The application provides a clean, intuitive interface for creating diagrams with a focus on usability and aesthetics.

Users can drag clean shapes without wrappers, visually connect them with arrows or lines, and leverage AI capabilities to generate diagrams directly from text prompts, streamlining the diagramming process for technical and non-technical users alike.

Key Features

  • Clean shape library without visual wrappers
  • Intuitive connection system
  • AI-powered diagram generation from text
  • Customizable styling options
  • Export in multiple formats
  • Collaborative editing
